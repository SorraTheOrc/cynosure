Cynosure

This is an open source bot that will help you stay coordinated and focussed. Essentially you can run standups with it and then review your objectives throughout the day.

The Bot will work with:

  • Cortana
  • Skype
  • Slack
  • Web Chat

To try out a demo of the application click the Skype icon (bottom right). You will need a skype account for this to work.

Check out the code on GitHub - contributions welcome.

Usage

Here's an example of the kind of conversation we hope to enable with Cynosure. At present this is not really how things work. As I type the conversation is more Question and Answer than conversation. This example is included to illustrate what is planned, you can see what the current status is by looking for and labels.

Start a standup

My main responsibility is to run your standup for you.

My commands:

‘start standup’ : Starts a new standup or continues and existing one.

‘Add to .’ : Add a single item to one of the available groups in the current standup.

‘Promote from .’ : Promote a single item from one of the available groups to the next “higher” group in the current standup.

‘Promote from .’ : Promote a single item from one of the available groups to the next “higher” group in the current standup.

‘Remove|Demote from .’ : Demotes a single item from one of the available groups in the next “lower” group in the current standup.

‘Delete from .’ : Deletes a single item from the current standup.

‘standup summary’ : Display a summary of the current standup.

‘help’ : Get hints on what is possible with this bot.
