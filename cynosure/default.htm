Cynosure

This is an open source bot that will help you stay coordinated and focussed. Essentially you can run standups with it and then review your objectives throughout the day.

The Bot will work with:

  • Cortana
  • Skype
  • Slack
  • Web Chat

To try out a demo of the application click the Skype icon (bottom right). You will need a skype account for this to work.

Check out the code on GitHub - contributions welcome.

Usage

Start standup
Start a standup. Answer the questions in order. Say "Nothing else" to end input on a question.
Report
Report on current standup
Help
Get help
Version
Get version information for this instance of the bot
